# TogetherLion
같이사자어플 개발(RN)

## 서비스 소개
물건을 공동구매하거나 택시를 같이 탈 사람을 구하는 수요가 커뮤니티에 늘어남에 따라 우선 금오공과대학교 학생들을 대상으로 물건을 공동구매 또는 배달을 같이 시킬 사람을 모집할 수 있는 기능을 제공하여
배달비를 아낀다. 추가로 택시 카풀할 사람을 모집할 수 있다.
